Firmware Overview
This hackpad uses KMK firmware.

Currently, the encoder is for volume, and the keys do simple f7-f9 functions. Once I build this up in person I am going to make a whole mini OS in it that will help me learn python with CircuitPython. This will be a challenge on such a small screen which will make this project even more interesting and challenge me. This firmware will include the following:

- Home page/menu
- Study mode
      - Adjustable pomodoro timer
      - Total study hours accumulator
- Food/Drink reminders
- Programmable RTC
- BLE connection to laptop/phone


BOM:

- 3x Cherry MX Switches
- 3x DSA Keycaps
- 4x M3x5x4 Heatset inserts
- 3x M3x16mm SHCS Bolts
- 1x 0.91" 128x32 OLED Display
- 1x EC11 Rotary Encoder
- 1x XIAO RP2040
- 1x Case (top and bottom)
